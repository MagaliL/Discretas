Ejercicio 8. Para los valores n = 54 y d = 4 encuentre los enteros q y r tales que:

n = dq + r


0 ≤ r < d



Ejercicio de: Epp, S. S. (2012). Matemáticas Discretas Con Aplicaciones (4ta ed.). Cengage Learning.

Objetivo: Descomponer un número entero n en términos de un divisor d comprendiendo la relación entre cociente, divisor y residuo.

Se requiere encontrar los enteros que cumplan con la expresión: n = dq + r con 0 ≤ r < d, para lo que es necesario sustituir los valores indicados: n = 54 y d =4.

n = dq + r

54 = 4q + r

Para hallar d y r se descompone 54 en el producto de dos factores, uno de los cuales es el 4. Por lo tanto, se divide 54 entre 4 para obtener d siendo r el residuo requerido en la expresión n = dq + r.



Entonces, el cociente 13 será d y el residuo r es 2, que cumple la condición: 0 ≤ r < d.

54 = 4*13 + 2


54 = 52 + 2


Por lo tanto, se tiene que:

54 = 4*13 + 2 con q= 13 y r= 2





Fin del ejercicio.
