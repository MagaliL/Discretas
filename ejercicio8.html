Ejercicio 8. Para cada uno de los siguientes valores de n y d, encuentre los enteros q y r tales que n = dq + r y 0 ≤ r < d.
n = dq + r


0 ≤ r < d



a) n = 54, d = 4
b) n = -54, d = 4
c) n =54, d = 70


Ejercicio de: Epp, S. S. (2012). Matemáticas Discretas Con Aplicaciones (4ta ed.). Cengage Learning.

Para la solución de este ejercicio se necesita revisar cada uno de los incisos que se proponen como solución:

a) n = 54, d = 4     b) n = -54, d = 4     c) n =54, d = 70


a) Si se requiere encontrar los enteros que cumplan con la expresión: n = dq + r con 0 ≤ r < d, sustituiremos los valores que nos da el inciso a: n = 54 y d =4.

n = dq + r

54 = 4q + r

Para encontrar d y r es necesario descomponer al 54 en el producto de dos factores, uno de los cuales es el 4. Por lo tanto:

54 = 4q + r

se debe tener en cuenta que r va de 0 a 4

54 = 4*13 + 2


Por lo tanto, para el inciso a se tiene:

54 = 4*13 + 2, con q= 13 y r= 2



b) Si se requiere encontrar los enteros que cumplan con la expresión: n = dq + r con 0 ≤ r < d, sustituiremos los valores que nos da el inciso b: n = -54 y d = 4.

n = dq + r

-54 = 4q + r

En este caso se tiene que tomar en cuenta que r está en el rango de 0 a 4, pero al ser n negativa la dinámica cambia.

-54 = 4*(-14) + r

Para conservar el signo negativo se requiere multiplicar por un número que nos de una cifra mayor a n

-54 = -56 + r

-54 = -56 + 2


Por lo tanto, para el inciso b se tiene:

-54 = 4*(-14) + 2, con q= -14 y r= 2



c) Si se requiere encontrar los enteros que cumplan con la expresión: n = dq + r con 0 ≤ r < d, sustituiremos los valores que nos da el inciso c: n = 54 y d = 70.

n = dq + r
54 = 70q + r
54 = 70*(0) + r
54 = 0 + r
54 = 0 + 54

Por lo tanto, para el inciso c se tiene:

54 = 70*(0) + 54, con q= 0 y r= 54
