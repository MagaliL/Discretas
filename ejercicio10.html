Ejercicio 10.

Cifrar el mensaje "OPENAI" utilizando el algoritmo RSA. Usar los siguientes valores para generar las claves y realiza el cifrado con:

p = 17
q = 19



Objetivo: Desarrollar la habilidad para aplicar conceptos de aritmética de residuos en el contexto de la criptografía, específicamente en el algoritmo RSA, y comprender su papel en la cifrado de mensajes.

Generación de claves:


Para comenzar a resolver este problema es necesario calcular el valor de n y φ(n) y seleccionar un valor adecuado de e.

p = 17, q = 19
n = p * q → 17 * 19 = 323
φ(n) = (p-1) * (q-1) → 16 * 18 = 288


Ahora se selecciona un valor para e. Elegimos e = 5; ya que es primo y coprimo con 288.

Por lo tanto, la clave pública es (e, n) = (5, 323).


Cifrado del Mensaje "OPENAI"


Obtener los valores de cada letra de la palabra en código ASCII.

O = 79, P = 80, E = 69, N = 78, A = 65, I = 73


Se utiliza cifra cada letra con el módulo tomando la clave pública (5, 323)

C ≣ Me (mod n)

Donde C es el mensaje cifrado.


O = 795 (mod 323) ≣ 58
P = 805 (mod 323) ≣ 275
E = 695 (mod 323) ≣ 258
N = 785 (mod 323) ≣ 295
A = 655 (mod 323) ≣ 77
I = 735 (mod 323) ≣ 58

El mensaje cifrado es: (58, 275, 258, 295, 77, 58)
