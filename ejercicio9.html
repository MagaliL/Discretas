Ejercicio 9. Bob quiere enviar el mensaje "HOLA" a Alicia utilizando el algoritmo RSA. El cifrado se realizará utilizando los valores ASCII de las letras del mensaje. Los valores de las letras y su correspondiente código ASCII son:


H = 72, O = 79, L = 76, A = 65


Para este ejercicio, se utilizan los siguientes valores de RSA:

p = 17
q = 19
e = 5


Objetivo: Desarrollar la habilidad para aplicar conceptos de aritmética de residuos en el contexto de la criptografía, específicamente en el algoritmo RSA, y comprender su papel en la cifrado de mensajes.


Generación de claves:


Para comenzar a resolver este problema es necesario calcular el valor de n y φ(n)

p = 17, q = 19
n = p * q → 17 * 19 = 323
φ(n) = (p-1) * (q-1) → 16 * 18 = 288


Ya se cuenta con e = 5 que es primo y coprimo con 288.

Por lo tanto, la clave pública es (e, n) = (5, 323).


Cifrado del Mensaje "OPENAI"


Obtener los valores de cada letra de la palabra en código ASCII.

H = 72, O = 79, L = 76, A = 65


Se utiliza cifra cada letra con el módulo tomando la clave pública (5, 323)

C ≣ Me (mod n)

Donde C es el mensaje cifrado.


H = 725 (mod 323) ≣ 21
O = 795 (mod 323) ≣ 129
L = 765 (mod 323) ≣ 247
A = 655 (mod 323) ≣ 12



El mensaje cifrado es: (21, 129, 247, 12)
