Ejercicio 7. Determine el inverso modular para
3 (mod 5).

Objetivo: Encontrar el inverso modular de un número en el contexto de la aritmética modular, ya que una de sus principales aplicaciones está en la criptografía.

Para obtener el inverso modular se debe recordar que para a (mod c) se debe encontrar una b que al multiplicarse por a dé como resultado 1.

a*b (mod c) = 1


Calcular a*b (mod c) para una b con valores entre 0 y c-1

a = 3
c = 5
c-1 = 4


3*0 ≡ 0 (mod 5)
3*1 ≡ 3 (mod 5)
3*2 ≡ 1 (mod 5) -> Este es el inverso modular
3*3 ≡ 9 ≡ 4 (mod 5)
3*4 ≡ 12 ≡ 2 (mod 5)
